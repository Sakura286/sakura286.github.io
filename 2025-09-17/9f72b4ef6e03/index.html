有时候我们想要更灵活地处理 Debian 的打包脚本，使得我们可以通过参数来影响 control/install 之类的文件，那么应该怎么办呢？

1. install 的处理

可以在 install 文件里留一个占位符，然后在 rules 文件里用sed进行替换

*.install:

rules:

将文件从 debian/tmp 安装到 debian/${pkg_name}，是发生在 dh_install 阶段的。所以只要在 dh_install 之前完成替换即可。
